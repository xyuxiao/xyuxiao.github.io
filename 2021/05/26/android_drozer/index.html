安装方法

  1. 下载选择drozer(Python.whl):https://labs.f-secure.com/tools/drozer/
  2. 安装drozer
  • 安装

    sudo pip install drozer-2.4.4-py2-none-any.whl

  • 安装完成, 需要twisted插件

    sudo pip install twisted

  1. 下载选择drozer (Agent .apk only)并给手机安装
  2. 使用drozer命令
    • 端口转发

      adb forward tcp:31415 tcp:31415

    • 连接控制台

      drozer console connect
