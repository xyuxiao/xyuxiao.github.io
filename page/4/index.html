编译源码Mac

发表于 2016-11-14 | 更新于 2021-06-08 | 分类于 Android

Mac

下载源码

  1. 配置环境
    • MacOS10.11.6
    • 谷歌文档http://source.android.com/source/initializing.html
    • 新建大小写区分的磁盘: hdiutil create -type SPARSE -fs ‘Case-sensitive Journaled HFS+’ -size 100g ~/android.dmg
    • 挂载磁盘: hdiutil attach ~/android.dmg.sparseimage -mountpoint /Volumes/android;
    • 修改~/.bash_profile，添加# mount the android file image function mountAndroid { hdiutil attach ~/android.dmg -mountpoint /Volumes/android; },之后可以直接输入$ mountAndroid挂载
    • 安装jdk
  2. 从清华大学镜像站下载AOSP,地址:https://mirrors-i.tuna.tsinghua.edu.cn/help/AOSP/
  3. 进到aosp目录下执行
    • 设置环境: $ source build/envsetup.sh 或 $ . build/envsetup.sh
    • 选择编译对象: $ lunch
    • 编译: $ make -jn

新建swap分区

  1. 建立一个img文件，count是大小: dd if=/dev/zero of=swap.img bs=1024 count=500000
  2. 格式化: mkswap swap.img
  3. 激活: sudo swapon swap.img
