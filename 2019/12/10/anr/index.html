ChkBugReport

下载

源码
点击下载

  1. 通过命令获取bugreport文件

    adb bugreport

  2. 拿到bugreport.zip解压,拿到里面的txt

    java –jar chkbugreport-0.5-215.jar bugreport-2019-12-10-16-52-43.txt

  3. 通过浏览器打开/bugreport_out/index.html

使用Battery Historian工具分析Android

下载

点击下载

使用

  1. 搜索Battery Historian的镜像

    docker search battery

  2. 运行battery-historian镜像

    docker run –name=battery -d -p 9999:9999 bhaavan/battery-historian

  3. 浏览器打开http://localhost:9999/

docker使用

  1. 查看docker里运行的服务

    docker ps -a

  2. 关闭运行的服务(battery是服务名称)

    docker rm battery
