新建项目

根据官方GitHub(https://github.com/cocos2d/cocos2d-x)进行配置,当执行新建项目的时候遇到

cocos new MyGame -p com.xyuxiao.mygame -l cpp -d ../Project/

  • -p 包名 -l 语言版本 -d 项目路径

-bash: cocos: command not found
//把配进环境变量就行了
open ~/.bash_profile
export PATH=/Users/xyuxiao/Documents/study/cocos/cocos2d-x/tools/cocos2d-console/bin:$PATH
source ~/.bash_profile
