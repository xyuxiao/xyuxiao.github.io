HttpClient关于SDK>23

Android23开始把HttpClient删掉了, 在这之后导入HttpClient会提示找不到, 可以在当前module的build.gradle中加入

Android8出现app is in background uid UidRecord

Android9出现requires android.permission.FOREGROUND_SERVICE

java.lang.SecurityException: Permission Denial: startForeground from pid=9564, uid=12350 requires android.permission.FOREGROUND_SERVICE

禁止录屏

运行项目报错Error while executing: am start -n “xxxxxx” -a android.intent.action.MAIN -c android.intent.category.LAUNCHER

运行项目出现以上报错, 搜了谷歌给出的解决方案

  1. clean, Invalidate Caches / Restart
  2. 关闭Instant Run

第一种方法尝试没有效果, 第二种方法AS用的是4.0, 已经没有Instant Run了。最后发现是Edit Configurations里Installation Options里Deploy选了Nothing,改为Default APK再运行就解决了问题。
