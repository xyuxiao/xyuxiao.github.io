adb

发表于 2019-04-29 | 更新于 2019-06-21 | 分类于 Android
  1. 抓取实时log: adb logcat
    • adb logcat -v time 打印结果带上时间
    • adb logcat *:E 打印Error日志
    • adb logcat -s DealData:E 打印tag为Error的Error日志
    • adb logcat > E:\log.txt 打印日志并保存到log.txt
  2. 唤醒屏幕: adb shell input keyevent 26 (26电源键)(模拟按键)
  3. 清除数据: adb shell pm clear 包名
  4. 安装apk: adb install 包名 (强制安装 adb install -r 包名)
  5. 查看当前栈顶的Activity: adb shell dumpsys activity | findStr “mFocusedActivity”
  6. 查看当前Activity栈:adb shell dumpsys activity | findStr -i run
