显示SystemUI

  • frameworks/base/services/java/com/android/server下的SystemServer.java的

http://blog.csdn.net/zwlove5280/article/details/48349331

AndroidManifest.xml

  • SystemUI属于系统apk

    android: sharedUserId=”android.uid.systemui”

  • SystemUI属于核心apk

    coreApp=”true”

  • SystemUI不会被KILL，即被KILL后会自动重启,主要指不被low memory killer杀掉

    android:persistent=”true”

  • 进程名为com.android.systemui

    android:process=”com.android.systemui”
